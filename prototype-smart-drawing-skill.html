智慧绘图技能页 · 改造说明

1去掉模型选择：顶部导航栏不再展示模型选择器（如"文心4.5 Turbo"），左侧侧导保持原有频道 Tab。侧导可展开/收起——展开时顶导：Logo + 收起按钮；收起时顶导：Logo + 展开按钮 + 新建对话。右侧下载 + 通知始终可见。
2去掉图片上传入口：输入框内去掉"+ 图片"上传按钮以及 placeholder 中的"或上传参考图"文案。
3去掉功能展示区：输入框与 Showcase 之间不再展示"风格模仿、风格转换、背景替换、局部重绘、一键消除"等 5 个功能卡片。
4新增图片比例选择：输入框左下角增加比例选择器，提供 1:1 / 3:4 / 4:3 / 9:16 / 16:9 / 2:3 / 3:2 七种比例，选择后 Showcase 区域同步切换展示对应比例的图片。
5Showcase 灵活比例：展示区固定 4 张图，自适应当前选中比例（如 9:16 竖图、16:9 横图等），不再固定 1:1 正方形。hover 显示 prompt + 做同款按钮。
⬅ 与频道页区别：此页面为「智慧绘图技能页」，不同于频道页（瀑布流+分类Tab）。技能页聚焦单次生图体验：输入 → 比例 → 生成，底部展示 4 张示例图作为灵感参考。
